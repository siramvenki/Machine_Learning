01. Results Report
    Step By Step Guide                                                                [Top] [Back to Deliverable List]


1. Read the email from Ajit Singh carefully to prepare for this task.

2. Download and review the template from the attachment to the email. It describes specifically what kind of results you should report.

3. Review the essential reading referenced in Readings & Resources that introduces concepts of data mining which are necessary to start the project.

4. In this project, the companies are categorized into 5 classes. Each class has its preferred services. You have to figure out which classes the new companies belong to, so as to make a better decision on the selling strategies via the data mining technique. Make sure to read the data explanation, and that you understand the meaning of each input item.

5. Write a Java program to implement the kNN algorithm. In this program, you should load the input file, preprocess the data (1 - deal with incomplete information; 2 - convert the symbolic/nominal attributes into continuous attributes via similarity matrix; and 3 - scale continuous attributes into the range of [0,1], using (xi - xi_min)/(xi_max - xi_min), where xi_max and xi_min are the maximum and minimum value for the ith attribute in the training set), and predict the behavior of new customers based on the knowledge of existing customers. TrainProdSelection.arff contains the data for existing customers, and testProdSelection.arff contains the data for new customers. Use k =3 as the initial results. Refer to the kNN pseudo-code and the algorithm description in Readings & Resources, which helps you to get started.

6. The following two records are provided to verify your implementation of kNN. If your kNN algorithm is implemented correctly, the output will be the same as below:

Type Life Style Vacation eCredit Entry Salary Property Class
Librarian spend>>saving 48 35 20 2 C4
Student spend<<saving 10 15 12 3 C1

7. Estimate the accuracy of kNN algorithm. Use the Jackknifing methodology to estimate the expected prediction accuracy of your results. Refer to the ‘Description of kNN algorithm’ document for details on the usage of the algorithm. Your accuracy should be above 90%.

8. Fill out the result report template based on the results from your development of the Java program.

9. Download the WEKA data mining software following the downloading instruction. Launch the WEKA GUI interface, load the data for existing customers into the system, load the data for new customers, generate the decision tree and predict the new customers’ data according to the tree. Use the default setting in this case.

10. Fill out the result report template based on the generated decision tree and prediction results.

11. Change the value of confidenceFactor for the decision tree. Check if you can find better result in terms of training accuracy.

12. Fill out the result report template based on the generated decision tree, your best training accuracy and the corresponding parameters.

13. Save the completed result report as a Microsoft Word document, named <<ProductSelection_Result_Report.doc>>.

14. Submit the report to your Program faculty for review.
