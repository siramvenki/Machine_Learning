Task 11 Data Mining Skill List
 

You are expected to acquire the following technical skills by the end of this task.  Everyone must master all of the skills.  You will be evaluated individually to see how well you have done.

1. Learn about data mining techniques, kNN, similarity matrix, decision trees and others

2. Understanding and interpreting confidence factors regarding data mining and analysis.

3. Integrating data mining techniques into launching a new products for a company.
